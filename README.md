# graph3d

## About graph3d

plotting 3-dimension data

![UI example](https://github.com/Tee0125/graph3d/blob/master/graph3d.png?raw=true)

## Usage

```
g = Graph3D()

g.add_point((1.0, 1.0, 1.0))
g.add_point((1.5, 1.5, 1.5))
g.add_point((2.0, 2.0, 2.0))

g.add_point((-1.0, -1.0, -1.0))

g.show()
```
